### 1. 国际化（R054）- P0 必需

**前端**：

- 语言包：`en.json` / `zh-CN.json`
- 支持动态切换

**后端**：
- AI 提示词多语言版本
- 根据用户偏好动态加载
- 工具描述、错误消息国际化

**工作量**：5 天

---

### 2. 营销首页（R055）- P0 必需

**页面结构**：
- Hero Section（首屏吸引）
- Features（特性展示）
- Pricing（定价说明）
- Footer（法律文档链接）

**技术方案**：
- Next.js 独立项目
- 部署到 Vercel/Netlify
- 响应式设计
- SEO 优化

**工作量**：5 天

---

### 3. 付费订阅系统（R057）- P0 必需

**套餐设计**：

| 套餐 | 价格 | AI 调用 | 存储 | 并发任务 |
|------|------|---------|------|---------|
| 免费试用 | $0 | 100 次 | 100MB | 1 |
| 个人版 | $9.99/月 | 1000 次 | 1GB | 3 |
| 专业版 | $29.99/月 | 5000 次 | 10GB | 10 |
| 企业版 | 定制 | 无限 | 无限 | 无限 |

**技术实现**：
- Stripe 支付集成
- 配额管理中间件
- 数据库表：subscriptions、usage_records、plans
- Webhook 处理订阅状态

**工作量**：5 天

---

### 4. 用户认证增强（R056）- P1 高优先级

- 第三方登录（可选：Google/GitHub）

**工作量**：3 天

### 5. 用户仪表板（R058）- P1 高优先级

**功能模块**：

## 🛠️ 技术栈

### 前端
- React 19 + TypeScript
- react-i18next（国际化）
- Stripe React（支付界面）
- recharts（图表）
- Tailwind CSS

### 后端
- Node.js + Express
- Stripe Node SDK
- SQLite/MySQL
- nodemailer（邮件）

### 基础设施
- Vercel/Netlify（首页）
- 自有服务器（主应用）
- Cloudflare（CDN + DNS）
- Sentry（错误监控）
